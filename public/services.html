9つのセミナー

FILM学習理論とチーム学習に基づく体系的プログラム

クリエーティブクエスチョン

問いが変わる。創造が始まる。

固定観念を打ち破る問いの技術を学び、創造的思考力を身につけます。問いの質が答えの質を決める。新しい視点と解決策を生み出す力を養います。

2日間 / オンライン対応 詳細を見る

人を育てるリーダーシップ

命令ではなく、成長を。

メンバーの自律性と成長を促進するリーダーシップスタイルを習得。支配ではなく支援、命令ではなく質問を通じて、真のリーダーシップを発揮する方法を学びます。

2日間 / オンライン対応 詳細を見る

コミュニケーションの本質

伝えるから、つながるへ。

表面的なテクニックを超えた、心と心をつなぐコミュニケーションの真髄を探求。聞く力、伝える力、そして相互理解を深める実践的スキルを身につけます。

2日間 / オンライン対応 詳細を見る

GoodSelfイメージ統合法

理想の自分に、なる。

理想の自分と現在の自分を統合し、自己受容と成長のバランスを取る方法を学習。セルフイメージの変革を通じて、真の自信と行動力を身につけます。

2日間 / オンライン対応 詳細を見る

手段目的ゲーム

目的を見失わない。

手段と目的を明確に区別し、本質的な目標達成のための思考法を習得。複雑な状況でも本来の目的を見失わず、効果的な解決策を見つける力を養います。

1日間 / オンライン対応 詳細を見る

メタ記憶術

記憶を使いこなす。

単なる記憶テクニックを超えた、記憶のメカニズムを理解し活用する方法を学習。効果的な学習法と情報整理術で、知識を智慧に変換する力を身につけます。

1日間 / オンライン対応 詳細を見る

次につながる振り返りの技術

経験を、智慧に。

経験から最大限の学びを抽出し、次の行動に活かす振り返りの技術を習得。単なる反省を超えて、成長と改善につながる建設的な振り返りのプロセスを学びます。

1日間 / オンライン対応 詳細を見る

読書が楽しくなる読み方

読むから、対話するへ。

受動的な読書から能動的な対話へ。本と著者との対話を通じて深い理解と洞察を得る読書法を習得。読書の効果を最大化し、実生活に活かす方法を学びます。

1日間 / オンライン対応 詳細を見る

メンタルモデルとデシジョンメーキング

思い込みを、見抜く。

無意識の思考パターンと判断基準を明確化し、より良い意思決定を行う力を養成。メンタルモデルの理解を通じて、偏見や先入観を克服する方法を学びます。

2日間 / オンライン対応 詳細を見る

学習の進め方

基礎づくり

コミュニケーション力の向上

コミュニケーションの本質 人を育てるリーダーシップ
